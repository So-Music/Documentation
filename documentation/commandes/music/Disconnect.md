Supprime la file d'attente et déconnecte le bot du canal vocal

!!!info
Pour exécuter cette commande, vous devez être dans le même canal vocal que le bot.
!!!

### Utilisation

[/disconnect]()

### Permissions

+++ Utilisateur
Aucune autorisation n'est nécessaire
+++ Bot
Aucune autorisation n'est nécessaire
+++
