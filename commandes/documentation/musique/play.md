# Play

{% hint style="info" %}
Pour exécuter cette commande, vous devez être dans le même canal vocal que le bot.
{% endhint %}

## Utilisation

<mark style="color:blue;">/play (music)</mark> -> Nom ou URL de la musique

{% hint style="warning" %}
Sources musicales prises en charge:

* Spotify
* SoundCloud
* Mp3
* Radio (NRJ; SkyRock; KissFM; etc...)
* Twitch
{% endhint %}
